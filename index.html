  • Proyecto 2

    • Descripción

    Esta aplicación permite a los usuarios interactuar con un diccionario de palabras utilizando una estructura de árbol llamada Trie. Un árbol Trie (también conocido como árbol prefijo) es una estructura de datos jerárquica utilizada principalmente para almacenar y buscar palabras o cadenas de caracteres. En un árbol Trie, cada nodo representa una letra y cada camino desde la raíz hasta un nodo hoja forma una palabra. Las palabras se almacenan de manera que los caracteres comunes en la parte superior del árbol se comparten entre múltiples palabras, lo que reduce significativamente el espacio de almacenamiento requerido.
